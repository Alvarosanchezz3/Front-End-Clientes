Estructura de la aplicación:

Java

Utilizado para desarrollar el Back-End de la aplicación, gestiona la lógica empresarial y la interacción con la base de datos, en este caso MySQL.

Spring (Framework)

Utilizado para construir la API RESTful que sirve como interfaz entre el frontend y el backend. Spring facilita la gestión de dependencias, la configuración y proporciona funcionalidades como la gestión de transacciones.

MySQL

Empleado para almacenar y gestionar los datos de los clientes y otras entidades en la aplicación, proporcionando una estructura organizada y eficiente para la persistencia de datos de todos los clientes.

HTML / CSS

Empleado en el Front-End para definir la estructura básica de las páginas web y definir la estética, disposición y diseño de los elementos

TypeScript

Utilizado en el Front-End para la creación de los componentes de la aplicación, estos mejoran la calidad del código, facilitan el mantenimiento y proporcionan un desarrollo más robusto

Angular (Framework)

Empleado en el Front-End para crear una interfaz de usuario dinámica y reactiva. Angular facilita la manipulación del DOM, la gestión del estado de la aplicación y la comunicación efectiva con la API REST
